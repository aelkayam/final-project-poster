About 

The system recognizes Hebrew handwriting from an image, converts it to text, and corrects errors. The main goal is to help people with dysgraphia whose handwriting is unclear.

Python is a high-level, interpreted programming language that is widely used for web development, data analysis, artificial intelligence, scientific computing, and many other applications. It was created in the late 1980s by Guido van Rossum and released publicly in 1991.

Features

OCR

Handwriting recognition and conversion to text format in Hebrew.

Spell checking

Using a Hebrew dictionary to identify and correct spelling errors.

Editing

Create a document from the output and download it to your device.
